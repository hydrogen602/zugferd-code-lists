# ZUGFeRD 2.3.2 spec files

Download the package from [ZUGFeRD 2.3.2 spec files](https://www.ferd-net.de/en/downloads/publications/details/zugferd-232-deutsch), locate the code lists file `4. EN16931+FacturX code lists values v14 - used from 2024-11-15.xlsx`, and copy it into this directory.

After that, run
```sh
make remove-spaces
```
in the top-level project directory.
